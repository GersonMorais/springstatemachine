# spring statemachine

Este projeto tem como objetivo demonstrar o uso do Spring State Machine. Usei como base para estudos.

# Fonte de conhecimento

Para a criação deste projeto, foi utilizado como base o artigo [https://medium.com/nstech/spring-state-machine-como-op%C3%A7%C3%A3o-97144586bf48) do site Medium.

# Estudos
Para entender melhor o funcionamento do Spring State Machine, recomendo a leitura do artigo [Spring State Machine](https://www.baeldung.com/spring-state-machine) do site Baeldung.
